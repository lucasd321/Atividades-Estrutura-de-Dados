# Activities-Data-Structures
This repository contains solutions to fundamental programming exercises, focused on developing logic, array manipulation, repetition structures, and conditionals. Solutions are available in both C and Java.

The project is divided into five main activities, based on the proposed problems:

List 1:
- Grade Ranking: Processing a static vector of grades to calculate the arithmetic mean, count students above average, and find the highest grade without using ready-made sorting functions.
- ATM Simulator: Algorithm to calculate the minimum number of bills (R$ 50, R$ 20, R$ 10) for a given withdrawal amount, using division and remainder operators.
- Password Validator: Security checker that analyzes whether a string has at least 8 characters and one special character.

List 2:
- Time Converter: A program that receives an integer value in seconds and converts it to the detailed Hours:Minutes:Seconds format.
- Unique Element Checker: A function that iterates through an integer vector to identify if all numbers are distinct or if there are duplicates.
